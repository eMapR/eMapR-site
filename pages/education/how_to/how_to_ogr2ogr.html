ogr2ogr examples

The ogr2ogr command line program can be used to convert simple features data between file formats performing various operations during the process such as spatial or attribute selections, reducing the set of attributes, setting the output coordinate system or even reprojecting the features during translation.

Dependecies

  • GDAL

    • *Note that installing all of these pieces separately can be troublesome. The most straight forward way to make sure that you have everything you need is to install QGIS. The install comes with the OSGeo4W Shell, which allows you to run all GDAL/ORG utilities, including the Python-based ones. On top of that you get a great open-source GIS!

Examples

Convert from ESRI shapefile to GeoJSON while reprojecting

In a command prompt, terminal, or OSGeo4W Shell, CD to the directory where your image lives

cd /mock/demo/

Run ogr2ogr replacing the mock file names between the square brackets with your out and in file names (don't include the brackets)

ogr2ogr -f GeoJSON -t_srs EPSG:5070 [outFile.geojson] [inFile.shp]

Dissolve all features in a GeoJSON vector file

In a command prompt, terminal, or OSGeo4W Shell, CD to the directory where your GeoJSON file lives and enter the following command, replacing the mock file names between the square brackets with your out and in file names (don't include the brackets)

ogr2ogr -f GeoJSON [outFile.geojson] [inFile.geojson] -dialect sqlite -sql "SELECT ST_Union(geometry) as geometry FROM OGRGeoJSON"

In this command we use the -f (file format) argument to define GeoJSON as the out file format and the -dialect argument to define sqlite as the SQL dialect to use in our query. Note that "OGRGeoJSON" is the default layer name of a GeoJSON file. Get more info on this here

Apply a fixed buffer to an ESRI shapefile

To apply a buffer we use the ST_buffer function on the geometry field of the inFile shapefile

ogr2ogr -f "ESRI Shapefile" [outFile.shp] [inFile.shp] -dialect sqlite -sql "select ST_buffer(geometry, [buffer distance]) as geometry FROM [inFile]"  
                
Example:

                
ogr2ogr -f "ESRI Shapefile" [bufferedFile.shp] [origFile.shp] -dialect sqlite -sql "select ST_buffer(geometry, [1000]) as geometry FROM [origFile]"
