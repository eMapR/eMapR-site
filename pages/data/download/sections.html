Here are a few instructions to get you going.
  1. Browse data products by expanding the product layers using the icon and clicking on the icon to get the product description.
  2. Select data you want to download by checking the box next to the data layer name(s) of interest.
  3. Select a vector file to upload that defines the area you want data for. The selected data products will be clipped to this region. ESRI Shape file and GeoJSON formats are accepted. If SHP, you must include all associated files.
  4. Enter your email address, we'll email you with a download link when the data are ready
  5. Hit the submit button! The process could take a few minutes.

If you have questions, experience problems with the interface, or the data, please email Justin Braaten at jstnbraaten@gmail.com

Greatest Disturbance

Description

These data describe information regarding the greatest magnitude disturbance per pixel time series.

Layers
  • Year of detection - the year that a disturbance was first detected
  • Magnitude - the spectral delta of the disturbance
  • Duration - the number of years of vegetation decline
  • Pre-disturbance value - the spectral value before the disturbance was detected

Pixel Resolution

30 meters

Time Period

1984-2016
